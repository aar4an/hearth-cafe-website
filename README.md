# The Hearth Café — Restaurant Website

A fully responsive café website built using pure HTML, CSS, and vanilla JavaScript as part of a web development project.

## Features
- Home section with animated hero and highlights
- Interactive Menu with tabbed categories (Coffees, Drinks, Bites)
- About section with brand story and values
- Contact section with a working form and café details
- Mobile responsive with hamburger navigation
- Smooth scroll and hover animations

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Animations, Media Queries)
- Vanilla JavaScript (no frameworks or libraries)

## How to Run
Just open `hearth-cafe.html` in any browser — no setup needed.
